# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0.0] - 2026-05-02
### Added
- Complete mobile accessibility overhaul for Special Olympics athletes on iPads.
- Massive touch-friendly buttons on a fixed bottom toolbar for screens < 768px.
- Probability cloud rendering with Monte Carlo stats dashboard.
- Coach's Tip AI module for shot feedback.

### Changed
- Replaced mouse drag-and-drop logic with an accessible Tap-to-Place and Tap-to-Pickup mechanic.
- Added `touch-action: none` to the canvas to lock viewport scrolling during court interactions.

### Fixed
- Base kinematic friction engine using $\mu=0.05$ coefficient.
